# Estus(as close to backpack as possible)

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run index.ts
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.


Swaps are implemented using Jupiter’s Meta-Aggregator API. Development and testing of wallet functionality is performed on Solana Devnet. Since Jupiter Meta-Aggregator operates on Mainnet liquidity, successful swap execution requires a funded Mainnet wallet. The quote endpoint and API integration have been verified
